# Actividad_Sem9_Market_Frontend

---

## Descripción
Proyecto frontend desarrollado en **React** como parte de la actividad de la semana 9.

---

## Funcionalidades
- CRUD completo de productos (crear, leer, actualizar y eliminar)
- Categorías controladas del **1 al 5** mediante `select`
- Validación de formularios
- Control de stock con alertas visuales
- Estado activo / inactivo por producto

---

## Tecnologías
- React  
- CSS personalizado  
- API REST (Render)  
- PostgreSQL  

---

## Interfaz
- Diseño tipo dashboard moderno  
- Tabla dinámica de productos  
- Indicadores de stock bajo  
- Experiencia responsive y limpia
